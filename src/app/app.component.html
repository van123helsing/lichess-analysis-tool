Personalized opening explorer for Lichess platform.
It uses the Lichess API to download games of a given player and generates a book of openings. Downloading all games takes some time due to the Lichess API limit, but once you have the .PGN file you can load it from your computer.
This is still a work in progress. Source code is available at GitHub. Any suggestions for improvment or found bugs can be report under Issues
